To learn more about the release, try:

  $ helm status {{ .Release.Name }} -n {{ .Release.Namespace }}
  $ helm get values {{ .Release.Name }} -n {{ .Release.Namespace }}
  $ helm get all {{ .Release.Name }} -n {{ .Release.Namespace }}

## Get Started

1. Check deployment status:
   ```
   kubectl rollout status deployment/{{ include "trading-engine.fullname" . }} -n {{ .Release.Namespace }}
   kubectl get pods -n {{ .Release.Namespace }} -l app={{ include "trading-engine.name" . }}
   ```

2. Port-forward to the service:
   ```
   kubectl port-forward -n {{ .Release.Namespace }} svc/{{ include "trading-engine.fullname" . }} 8080:{{ .Values.service.port }}
   ```

3. Test the API:
   ```
   curl http://localhost:8080/health
   curl http://localhost:8080/metrics
   ```

4. View logs:
   ```
   kubectl logs -n {{ .Release.Namespace }} -l app={{ include "trading-engine.name" . }} -f
   ```

## Monitoring

Watch HPA scaling:
```
kubectl get hpa -n {{ .Release.Namespace }} -w
```

View metrics:
```
kubectl port-forward -n prometheus svc/prometheus 9090:9090
# Visit http://localhost:9090
```

## Upgrade

To upgrade the release:
```
helm upgrade {{ .Release.Name }} infrastructure/helm/ \
  -n {{ .Release.Namespace }} \
  --values values.yaml
```

## Rollback

To rollback to previous version:
```
helm rollback {{ .Release.Name }} -n {{ .Release.Namespace }}
```

## Uninstall

To uninstall the release:
```
helm uninstall {{ .Release.Name }} -n {{ .Release.Namespace }}
```
